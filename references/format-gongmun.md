# 시행문 (Gongmun) 작성 가이드

> 공공기관 보고서 작성 강의자료 p.14-15 + 행정업무의 운영 및 혁신에 관한 규정(시행규칙 별표)

---

## 1. 핵심 철학

> **보고된 사항을 내·외부에 공지·지시하기 위한 공식 문서.**
> 보고서가 "현황 파악·내부 의사결정"이라면, 시행문은 "결정사항의 실행 알림".

독자가 **타부서·외부기관·일반국민**이므로:
- 서술식 + 개조식 **혼용**
- 격식 갖춘 결문(`~하기 바랍니다`)
- 발신명의·수신·일자 **모두 명시**

---

## 2. 표준 골격 (행정안전부 표준 기반)

```
┌──────────────────────────────────────────────┐
│        [발신기관 로고 또는 명칭]                  │
│                                              │
│ 수    신: 수신자 (예: 각 부서장)                  │
│ (경    유):                                  │
│ 제    목: 제목 (간결한 명사형)                    │
│                                              │
│ [본문 시작]                                    │
│                                              │
│  1. 관련: 근거 (법령·계획 등) 명시 (서술식 1–2문장)  │
│                                              │
│  2. 위 호 관련 ~ 다음과 같이 알려드리니             │
│     적극 협조하여 주시기 바랍니다. (서술식)            │
│                                              │
│   가. 첫 번째 안내사항 (개조식)                     │
│   나. 두 번째 안내사항                            │
│   다. 세 번째 안내사항                            │
│                                              │
│  붙임  1. 관련 자료 1부.                         │
│        2. 관련 자료 1부.   끝.                   │
│                                              │
│                                              │
│              발신 부서장 직위 + 직인                │
│                                              │
│  수신자: ~                                    │
│  기안자: 직급 ○○○   검토자: ~   협조자: ~        │
│  시행 부서명-일련번호 (날짜)   접수 ~              │
│  부서주소  / 부서전화 / fax / 메일 / 공개구분         │
└──────────────────────────────────────────────┘
```

## 3. 필수 구성 항목

| # | 항목 | 위치 | 필수 |
|---|------|------|------|
| 1 | 수신 | 상단 | ✓ |
| 2 | (경유) | 수신 다음 | △ (필요시) |
| 3 | 제목 | 수신 다음 | ✓ |

| 7 | 기안·검토·협조자 | 하단 | ✓ |
| 8 | 시행번호·일자 | 하단 | ✓ |
| 9 | 부서주소·연락처 | 최하단 | ✓ |

## 4. 항목별 작성 요령

### 4-1. 수신·경유·제목

- **수신**: `수신: 각 부서장 (○○부, ○○부, ○○부)` 또는 `수신: ○○유통 ○○○ 본부장`
- **(경유)**: 결재라인 중간 단계가 있으면 표시. 없으면 `(경유)` 만 두거나 생략
- **제목**: 명사형 + 간결, 25자 이내가 좋음
  - 예) `2025년 하반기 매장 운영실적 보고 요청`
  - 예) `「○○ 사업」 업무 협조 요청`

### 4-2. 본문

강의자료 패턴 (p.14):

> 시행문은 공지 혹은 지시의 목적으로 작성된 것으로 **주요내용은 서술식으로 기술하고, 세부정보에 한하여 개조식으로 작성**한다.

**구조 패턴**:

```
1. 관련: ~~~ (1줄 서술식, 근거 명시)

2. 위 호와 관련하여 ~~~을 다음과 같이 알려드리니
   적극 협조하여 주시기 바랍니다. (1–3줄 서술식, 본문 도입)

  가. 첫 번째 사항 (개조식)
  나. 두 번째 사항
    1) 세부 항목
    2) 세부 항목
  다. 세 번째 사항
```

### 4-3. 붙임 (별첨 자료가 있을 때)

```
붙임  1. 관련 계획서 1부.
      2. 양식(별지 제1호) 1부.   끝.
```

- 마지막 붙임 뒤에 **반드시** `끝.` 표기
- 붙임이 없으면 본문 마지막 문장 뒤에 `끝.`만 표기

### 4-4. 발신명의

본문 끝에서 두 줄 띄우고 가운데 정렬:

```
                      ○○주식회사
                  AI혁신처장 (직인)
```

또는

```
                      ○○공사 사장
                          (직인)
```

### 4-5. 하단 메타데이터

행정안전부 표준 양식에 따라 **세 줄 구조**:

```
기안자  AI혁신처 ○○과장 박○○      검토자  AI혁신처장 박○○      협조자  ○○처장 ○○○
시행  AI혁신처-1234 (2025. 11. 5.)                                접수
○○시 ○○구 ○○로 1                  /  02-0000-0000   /  fax 02-0000-0001  /  example@○○공사.com  /  대국민공개
```

## 5. 글머리 위계 (체계 B 강제)

시행문은 **반드시** 행정 위계를 사용:

```
1.  → 가.  → 1)  → 가)  → (1)  → (가)  → ①  → ㉮
```

문체는 다음을 준수:
- `~하기 바랍니다`, `~하여 주시기 바랍니다` (정중한 명령·요청)
- `~을 (다음과 같이) 알려드립니다` (공지)
- `~을 (다음과 같이) 통보합니다` (통보)

문장 끝은 모두 `다.` 또는 `다.` (반말 금지, 평어 금지).

| 제목 글꼴 | 함초롬돋움 14pt 굵게 |
| 줄간격 | 160% |
| 발신명의 글꼴 | 함초롬돋움 14pt |

---

## 7. 분량 가이드

시행문은 **간결성이 미덕**. 본문은 **A4 1쪽 안에 끝나는 것이 표준**.

- 1쪽 초과 시: 본문 압축 또는 별첨으로 분리 권고
- 본문 항목이 7개 초과 시: 가. 나. 다. 위계 정리 (2단계 들여쓰기)

---

## 8. 양식 미준수 위험 신호

- [ ] 발신명의 누락 → 빌드 거부
- [ ] 결문이 `~하시기 바랍니다` 패턴이 아님 → 자동 변환
- [ ] 제목이 25자 초과 → 단축 제안
- [ ] 본문에 `-적/의/것/들` 군더더기 → 자동 정리 (단, 공식 표현 보존)
- [ ] 글머리가 □○-* 위계로 작성됨 → 1.가.1)(1) 위계로 변환

## 9. 빌드 시 hwpx_helpers.py 호출 패턴

```python
parts.extend(make_gongmun_header(
    sender='○○주식회사 ○○○처',
    receiver='각 부서장',
    via='',  # (경유) 비어 있으면 생략
    title='2025년 하반기 매장 운영실적 보고 요청'
))

parts.append(make_gongmun_body(
    related_clause='2025년 매장 운영 평가 추진 계획(AI혁신처-1234, 2025.10.15.)',
    main_paragraph=(
        '위 호와 관련하여 매장별 운영실적 보고를 다음과 같이 '
        '요청하오니 적극 협조하여 주시기 바랍니다.'
    ),
    items=[
        '가. 보고 대상: 본사 직영 및 위탁 매장 전체',
        '나. 제출 항목',
        '  1) 매출 실적 (월별, 분기별)',
        '  2) 인력 운영 현황',
        '  3) 주요 이슈 및 개선 요청 사항',
        '다. 제출 기한: 2025. 11. 30.',
        '라. 제출 방법: 양식(별지 제1호)에 작성하여 메일 회신',
    ]
))

parts.extend(make_gongmun_attachment(
    files=['2025년 매장 운영실적 보고 양식 1부.']
))

parts.append(make_gongmun_signature(
    organization='○○주식회사',
    title='AI혁신처장'
))

parts.extend(make_gongmun_metadata(
    drafter=('AI혁신처 ○○과장', '박○○'),
    reviewer=('AI혁신처장', '박○○'),
    cooperator=('', ''),
    serial='AI혁신처-1234',
    date='2025. 11. 5.',
    address='○○시 ○○구 ○○로 1',
    tel='02-0000-0000',
    fax='02-0000-0001',
    email='example@○○공사.com',
    disclosure='대국민공개'
))
```

> 신규 함수 (hwpx_helpers.py 보강 대상):
> - `make_gongmun_header(sender, receiver, via, title)`
> - `make_gongmun_body(related_clause, main_paragraph, items)`
> - `make_gongmun_attachment(files)` — `붙임` 자동 + `끝.` 자동
> - `make_gongmun_signature(organization, title)` — 가운데 정렬, 직인 자리
> - `make_gongmun_metadata(...)` — 하단 3줄 메타정보
